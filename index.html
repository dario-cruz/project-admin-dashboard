Service Modules

Access Control List

List of rules that allow for the blocking of incoming and outgoing traffic. ACL rules are read from the top down, the lower the rule the more specific.

Reverse Proxy

A reverse proxy protects web servers from attacks and can provide performance and reliability benefits.

Ad Blocker

Block unwanted advertisements with a set of rules that stop spam and ads from entering the network. Superior option to browser extension based options.

VLAN Configuration

Create virtual segmented networks on a single physical network. Allows for more security as ACL rules can be applies per-vlan.

DNS Rules

Changes the way hostnames are resolved into IP addresses out on the internet. Custom DNS settings can be applied based on use preference.

Port Forwarding

A port forward is a way of making a computer on your home or business network accessible to computers on the internet, even though they are behind a router or firewall. It is commonly used in gaming, security cameras, home automation, and the Internet of Things (IoT).
